# 🚗 Vintage Cars Data Analysis

## 📌 Project Overview
This project performs Exploratory Data Analysis (EDA) on a vintage cars dataset.
The goal is to analyze car brands, models, price trends, and performance features.

- Source: Custom Dataset
- Format: CSV

- Pandas

- Price comparison
- Engine performance trends
- Geographic visualization using map
